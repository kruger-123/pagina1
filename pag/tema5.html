Protocolo TCP/IP

El Protocolo de Control de Transmisión/Protocolo de Internet (TCP/IP) es un conjunto de protocolos de comunicación que se utilizan para interconectar redes de computadoras en Internet.

1. TCP (Protocolo de Control de Transmisión): Este protocolo se encarga de asegurar que los datos se transmitan de manera correcta entre los dispositivos. Divide los datos en paquetes, los envía y luego los vuelve a ensamblar en el dispositivo receptor.

2. IP (Protocolo de Internet): Este protocolo se encarga de la dirección y el enrutamiento de los paquetes de datos. Cada dispositivo en Internet tiene una dirección IP única que se utiliza para enviar y recibir paquetes de datos.

Estos dos protocolos trabajan juntos para asegurar que los datos se transmitan de manera eficiente y segura a través de Internet.
